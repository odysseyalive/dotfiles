# Editor Tools Reference

## Tmux

### File Paths

| File | Purpose |
|------|---------|
| `tmux/tmux.conf` | Base tmux config |
| `tmux/plugin/` | Tmux plugins directory |
| `~/.yadrlite/tmux/plugin/` | Installed plugins location |

### Key Mappings

| Key | Action |
|-----|--------|
| `Ctrl-a` | Tmux leader key |
| `Ctrl-a Ctrl-s` | Save session (tmux-resurrect) |
| `Ctrl-a Ctrl-r` | Restore session (tmux-resurrect) |
| `Ctrl-h/j/k/l` | Vim-Tmux split navigation |
